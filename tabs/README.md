# 概要
Angular Materialを使用したタブの実装

# 環境
- node:　18.20.0
- Angular: 17.0.0
  
# 使用方法
## パッケージインストール
```
$ docker-compose run --rm angular npm install
```

## コンテナ起動
```
$ docker-compose up -d
```
ブラウザから localhost:4200/ にアクセス
