# 概要
Angular Materialを使用したハンバーガーメニューの実装

# 環境
```
node: 18.13.0
Angular: 15.0.0
```

# 使用方法
## パッケージインストール
```
$ docker-compose run --rm angular npm install
```

## コンテナ起動
```
$ docker-compose up -d
```

# 確認
ブラウザから localhost:4200/ にアクセス
